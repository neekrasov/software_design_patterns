# Прототип (prototype)

```
Порождающий паттерн проектирования, который позволяет копировать объекты, не вдаваясь в подробности их реализации.
Прототип - частично, или полностью инициализированный объект, который копируете (клонируете) и используете.
Дополнение:
Фабрика предоставляет удобный API для использования прототипов.
```
